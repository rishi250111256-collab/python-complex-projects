# Web Scraper (Python)

A simple Python web scraper that extracts quotes and authors from a website and stores them in a CSV file.

## Features
- Scrapes real website data
- Saves output to CSV
- Clean and beginner-friendly code

## Technologies Used
- Python
- requests
- BeautifulSoup

## How to Run
1. Install dependencies  
   pip install -r requirements.txt

2. Run the scraper  
   python scraper.py

## Output
- quotes.csv containing scraped data
